# simple-vpc-design
This project demonstrates the creation of a simple AWS VPC (Virtual Private Cloud) with a public subnet, internet gateway, and route table. An EC2 instance is launched inside the VPC to verify network connectivity. The project helps in understanding the basic concepts of AWS networking and cloud infrastructure.

# Project Overview

This project demonstrates the creation of a simple Virtual Private Cloud (VPC) in AWS. A VPC provides a secure and isolated network environment where AWS resources can be deployed and managed.

The project includes creating a VPC, public subnet, internet gateway, route table, and launching an EC2 instance inside the subnet.

# Services Used
Amazon VPC
Amazon EC2
Internet Gateway
Route Tables
# Architecture
Created a VPC with a custom CIDR block.
Created a Public Subnet inside the VPC.
Attached an Internet Gateway to the VPC.
Configured a Route Table for internet access.
Associated the Route Table with the Public Subnet.
Launched an EC2 instance in the Public Subnet.
# Steps Performed
Created a new VPC.
